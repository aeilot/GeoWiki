# 太阳系

太阳、行星及其卫星、矮行星和众多小天体等构成太阳系。太阳为中心天体。

## 八大行星

太阳系，各大行星按照与太阳的距离，由近到远依次为：水星、金星、地球、火星、木星、土星、天王星、海王星。

### 分类

八大行星可以分类为类地行星、巨行星和远日行星。

| 分类 |  |
| :----: | :----: |
| 类地行星 | 水星、金星、地球、火星 |
| 巨行星 | 木星、土星 |
| 远日行星 | 天王星、海王星 |


## 公转

在太阳的引力作用下，八大行星、矮行星以及众多的小行星、彗星等天体，都绕太阳公转。

以八大行星为例：从地球北极上空看，都按逆时针方向，公转轨道基本在同一个平面，且都为接近正圆的椭圆。所以总结出八大行星公转的三大特征：**同向性**、**共面性**、**近圆性**。
